Why this exists

A while ago, I was searching for a tool like this, but all of the ones I found were using an outdated Duckyscript version. The reason I needed this is due to how branded USB Rubber Duckys are INCREDIBLY expensive, so I got an Arduino Leonardo in USB form. It does the exact same thing as a Ducky, but it can only parse Arduino code. Hak5 provided so many payloads, but they were a pain to convert. That's how this tool was born.

How it works

This tool simpy takes provided Duckyscript commands and formats them to the Arduino equivalent. It's super simple and entirely written in Javascript. The code is available on the repo for my site.
